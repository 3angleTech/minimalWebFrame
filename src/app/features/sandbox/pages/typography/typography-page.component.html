Typography

Typography Classes

.typo-title-large

This is Title Large

 Use for large page titles, hero sections, or main headings.
.typo-title

This is Title

 Use for regular section titles or important headings.
.typo-subtitle

This is Subtitle

 Use for subtitles or supporting headings below titles.
.typo-body

This is standard body text

 Use for standard paragraphs or general content.
.typo-body-strong

This is Strong Body text

 Use for emphasized body text or important information.
.typo-body-small

This is Small Body text

 Use for smaller body text or secondary information, such as footnotes or disclaimers.
.typo-button Use for text inside buttons call-to-action elements.
.typo-caption

This is Caption text, used for image descriptions or annotations.

 Use for captions or small text below images or figures.
.typo-overline

OVERLINE TEXT

 Use for small uppercase text, often above headers or labels.
