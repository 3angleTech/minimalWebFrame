Text color - CSS Variables
--color-text-dark-enhanced

This is dark enhanced text

 Use for important or primary dark text (87% opacity).
--color-text-dark

This is dark text

 Use for regular dark text (60% opacity).
--color-text-dark-disabled

This is dark disabled text

 Use for disabled or low-importance dark text (30% opacity).
--color-text-light-enhanced

This is light enhanced text

 Use for important or primary light text (87% opacity).
--color-text-light

This is light text

 Use for regular light text (60% opacity).
--color-text-light-disabled

This is light disabled text

 Use for disabled or low-importance light text (30% opacity).

Typography - CSS Variables and Placeholders
--typo-scale-title-large
%typo-title-large

This is Title Large

 Use for large page titles, hero sections, or main headings.
--typo-scale-title
%typo-title

This is Title

 Use for regular section titles or important headings.
--typo-scale-subtitle
%typo-subtitle

This is Subtitle

 Use for subtitles or supporting headings below titles.
--typo-scale-body
%typo-body

This is standard body text

 Use for standard paragraphs or general content.
--typo-scale-body-strong
%typo-body-strong

This is Strong Body text

 Use for emphasized body text or important information.
--typo-scale-body-small
%typo-body-small

This is Small Body text

 Use for smaller body text or secondary information, such as footnotes or disclaimers.
--typo-scale-button
%typo-button
Use for text inside buttons call-to-action elements.
--typo-scale-caption
%typo-caption

This is Caption text, used for image descriptions or annotations.

 Use for captions or small text below images or figures.
--typo-scale-overline
%typo-overline

OVERLINE TEXT

 Use for small uppercase text, often above headers or labels.
